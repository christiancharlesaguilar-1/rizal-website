# Life and Works of Rizal - Educational Website

## 🇵🇭 About This Project

A comprehensive, historically-themed digital resource dedicated to **Dr. José Rizal**, the Philippines' foremost national hero. This website showcases his life, literary works, nationalist movements, martyrdom, and enduring legacy.

**Live Website:** [https://yourusername.github.io/rizal-website](https://yourusername.github.io/rizal-website)

---

## 📚 Website Sections

1. **Home** - Hero section with Rizal portrait and key statistics
2. **Biography** - Interactive timeline from 1861 to 1896
3. **Literary Works** - Noli Me Tángere & El Filibusterismo analysis
4. **Nationalism** - La Liga Filipina and Propaganda Movement reforms
5. **Martyrdom** - Final days, Mi Último Adiós, and execution details
6. **About Us** - Team information and project purpose
7. **Contact** - Working contact form

## 🚀 How to Deploy to GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [https://github.com](https://github.com) and sign in
2. Click the **+** icon in the top right → **New repository**
3. Name your repository: `rizal-website` (or any name you prefer)
4. Set it to **Public**
5. Click **Create repository**

### Step 2: Upload Your Files

**Option A: Using GitHub Web Interface (Easiest)**
1. In your new repository, click **"uploading an existing file"**
2. Drag and drop all files from the `rizal-website-github` folder:
   - `index.html`
   - `images/` folder with all images
   - `README.md`
3. Click **Commit changes**

**Option B: Using Git Command Line**
```bash
# Navigate to your project folder
cd rizal-website-github

# Initialize git
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit - Rizal website"

# Connect to GitHub (replace with your username)
git remote add origin https://github.com/YOUR_USERNAME/rizal-website.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click **Settings** (tab at the top)
3. Scroll down to **Pages** in the left sidebar (or click "Pages" directly)
4. Under **Source**, select **Deploy from a branch**
5. Under **Branch**, select **main** and folder **/(root)**
6. Click **Save**

### Step 4: Wait for Deployment
- GitHub will build your site (takes 1-5 minutes)
- Refresh the page to see the green confirmation
- Your site will be live at: `https://YOUR_USERNAME.github.io/rizal-website`
